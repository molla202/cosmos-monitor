# Changelog

## [0.1.0] - 2025-06-18

### Added
- Initial release
- Auto-detection of Cosmos chains from standard home directories
- Port auto-detection from `config.toml` and `app.toml`
- Support for Push Chain, Celestia, Lumera, Osmosis, Cosmos Hub, Axelar, Stride, Nolus
- Live TUI dashboard with Textual
- Per-chain colored tabs
- Node status panel (PID, uptime, memory, disk)
- Chain sync status with progress bar
- Network status (peers, latency, node ID)
- Validator status panel
- Paginated validator table
- Live log tail
- Oshvank ANSI logo branding
- One-line installer script
